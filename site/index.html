Open source spectrum reference data

More structure for radio data that people already share.

SSRF-Lite is an open source effort to add more structure, depth, and detail to the kinds of frequency and repeater information commonly found in RadioReference, RepeaterBook, club lists, agency pages, and local notes.

Readable by people

The source data stays in plain YAML so radio operators can review, correct, and contribute records without needing a proprietary tool.

Useful to software

Records separate organizations, sites, stations, RF chains, channel plans, tones, talkgroups, and authorizations so apps can build maps, codeplugs, documentation, and search views from the same data.

Built for provenance

SSRF-Lite can keep source links and notes alongside each data set, making it easier to understand where a record came from and what still needs verification.

Contributions are welcome

Add a missing system, improve a channel plan, correct a site, attach better source material, or help refine the schema. The goal is a practical shared library that gets better as more local knowledge is contributed.
